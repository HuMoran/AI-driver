# Python Rules

## Format
- Tool: `ruff format .`

## Lint
- Tool: `ruff check .`

## Test
- Command: `pytest`
- Coverage: `pytest --cov`

## Build
- Command: `python -m build` (if pyproject.toml exists)

## Project Structure
- src/<package>/: source code
- tests/: test files
- pyproject.toml: project config
